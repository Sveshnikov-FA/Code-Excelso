The Scala tutorial

Scala была разработан для того, чтобы объединить объектно-ориентированное и функциональное программирование в один язык высокого уровня. В основном она используется для разработки веб- и десктоп- приложений, а также для анализа данных. Возможно, сначала может показаться, что её освоение столь же достижимо, как покорение крутой скалистой горы, но это того стоит: синтаксис Scala легко читаем, и к тому же, она достаточно быстра. Чего ещё можно хотеть от подобного языка?
Для того, чтобы начать работать со Scala, необходимо убедиться, что установлено всё необходимое, как например, компилятор, который можно скачать на официальном сайте языка, и среду разработки, роль которой может выполнить Eclipse.

Уже стало традицией начинать свой опыт с языком выводом сообщения “Hello World” в консоль. Приступим же!

Код
Результат
def main (args: Array[String]): Unit = {
println("Hello, world!")
}

Разберём же код. Def – это стандартный способ объявления функций, о которых мы поговорим позже, main – это просто её название, аргументы перечислены в скобках, и её содержимое помещено в фигурные скобки. Функция main выполняется первой в программе. Unit – лишь базовое значение переменной, которую возвращает подобная функция, то есть ничего. Println – функция, которая вопреки ожиданиям… а, нет, всё-таки она отвечает лишь за вывод данных в консоль. Но стоит заметить, что в конце строк в Scala, в отличие от некоторых языков программирования, не ставится точка с запятой.
Теперь о переменных. Многие более современные языки программирования уходят от правил необходимости уточнения типа переменной, и Scala – не исключение. Разумеется, при желании это можно сделать, но это ни в коем случае не обязательно. Var используется для объявления переменных, а val – для объявления констант. Для того, чтобы создать массив, то есть набор однотипных элементов, помещённых в выделенные ячейки памяти, достаточно приравнять новую переменную к новому массиву, с уточнением размера и типа данных, как показано ниже. Для того, чтобы получить доступ к элементам нужно поместить его порядковый номер в массиве в квадратные скобки рядом с названием искомого массива.

var myVar = 10;
var myVar1 : String = "Foo"
var z = new Array[String](3)

Ввод данных пользователем – это основной способ обмена информации клиент - компьютер. Самый удобный способ получить его в Scala – использовать всеми любимый метод read*тип*() вот так:

var myVar = scala.io.StdIn.readInt()
var myV = scala.io.StdIn.readLine()

Переходим к условиям. Они достаточно понятны сами по себе: если условие выполняется, то выполняется и соответствующее действие. Выглядит это приблизительно так:

if(condition) {
do a thing
} else if (condition || condition) {
do a thing
} else if (condition && condition) {
do a thing
} else {
do a thing
}

Циклы – отличная вещь. Они делают нашу жизнь проще снова и снова. В каком бы языке, серьёзном, практичном, или литературном они ни были бы введены, они делают всё практичнее. В Scala существуют два типа циклов – for и while. С циклами for мы можем написать код, который должен быть выполнен конкретное количество раз, а с циклами while код будет выполняться пока выполняется заданное условие. Разумеется, есть способ изменить ход работы цикла, а именно – контрольные объявления, как например, continue, с котоым можно пропустить код, идущий после этого в цикле, и перейти к следующей итерации, и break, с которым выполнение цикла полностью прекращается.

while(condition) {
things to do
}
for( var x <- Range ){
things to do
}

Как мы помним, в любой, даже самой простой программе есть главная функция def main(). Давайте же рассмотрим функции поподробнее. Def – это ключевое слово, как var, для объявления функций, затем идёт название, затем – аргументы, значения переменных из внешего источника, посылаемые в данную функцию и принимающие соответствующее название переменной, помещённые в скобки. Теперь мы уточняем тип возвращаемых в место вызова данных через двоеточие. Потом идёт знак = , и только теперь, в фигурных скобках, мы пишем действия, которые мы хотим, чтобы функция выполняла.

def addInt( a:Int, b:Int ) : Int = {
var sum:Int = 0
sum = a + b
return sum
}

Давайте же сделаем проект. Веб-проект. Для этого нам понадобится lift, веб-фреймворк для scala. Для начала, введём sbt new lift/basic-app.g8 в консоль. Теперь откроем же файл .html в папке webapp, а также файл helloworld.scala. Посмотрим на последний внимательно. Теперь самое конструктивное в данной ситуации – это удалить всё содержимое класса. Так и поступим. Для запуска сервера, а соответственно, и для правильной работы сценариев страницы (которые мы только что удалили) достаточно ввести ~jetty:start в нашей sbt консоли. Одна из важнейших возможностей lift это возможность задать значение css и содержимого innerHTML элемента одной командой. Для получения значения HTML, которое нужно изменить, добавим атрибут data-lift к элементу со значением НазваниеКлассаВТомФайле.НазваниеФункцииКоторойНуженДоступКДанным. А в этот метод мы пишем “идентификатор” #> “Новое_значение”, где идентификатор может быть чем угодно: названием элемента, id, класс, и так далее! Для изменения атрибута, поместим его название в квадратные скобки после идентификатора in, для добавления к заданному значению поставим + после названия, и ! для удаления.

Это всё самое важное, что нужно знать о lift для начала. Поздравляю, мы покорили эту Скалу!
