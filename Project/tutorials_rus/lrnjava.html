The Java tutorial

Java – название сорта кофе, родом с одноимённого острова близ Индонезии. По чистой случайности, это ещё и язык программирования, выпущенный в 1995. Его в основном используют для разработки мобильных и десктоп приложений. Синтаксис языка несколько громоздкий, но область применения языка так или иначе крайне широка.
Для начала мы должны проверить, установлено ли у нас всё необходимое: компилятор JDK, доступный для скачивания на официальном сайте, и среда разработки, роль которой лучше всего выполняют IntelliJ IDEA и Eclipse IDE.

Для программистов уже стало традицией начинать изучать новый язык с вывода в консоль сообщения “Hello, World!”. Приступим же!

Код
Результат
import java.util.*;
class Main {
public static void main(String[] args) {
System.out.println("Hello world!");
} }

Давайте разберём этот код. Import – ключевое слово для импортирования дополнительных библиотек в проект. Class Main – главный класс главного файла, с главной функцией main(String[] args) внутри, определение которой должно выглядеть так, как показано выше, и никак иначе. Public означает, что эта функция видна всем файлам и классам проекта, static - что она не изменит свойства до конца выполнения программы, void означает, что это функция, которая не возвращает значения, main – её название, а в скобках перечислены аргументы. Их может быть много, может не быть совсем. Они объявляются прямо как переменные (о них попозже). System.out.println() отвечает за печать в консоль на новой строке. Если мы хотим вывести число или переменную, то кавычки для этого не нужны. Можно попробовать сломать стереотипы и вывести в консоль данные двух типов. Тогда они сиоединены в println знаком +. Также есть сверхсекретные коды которые используют только программисты с большой буквы ПРО, но я поделюсь этими знаниями. “\n” переводит на новую строку, “\b” удаляет последний символ вывода, “\”” позволяет вставлять двойные кавычки в вывод, ведь если бы мы этого не сделали, он бы закончился на них.
Далее о переменных. Они в Java есть, поэтому стоит их упомянуть. Int и long - целые числа соответствующего размера, float и double – действительные числа, где double может быть больше, чем float, char – одиночный символ. String - это много символов, а Boolean принимает значения только true или false. Стоит заметить, что в Java переменные аб и АБ – разные переменные.
Принимать пользовательский ввод достаточно просто с помощью модуля Scanner, который сначала нужно импортировать, а затем объявить, и только после этого им можно будет пользоваться.

import java.util.*;
class sample {}
class Main {
public static void main(String[] args) {
Scanner s = new Scanner(System.in);
int a = s.nextInt();
} }

Далее об условиях. Они работают достаточно очевидным образов, вытекающим из их определения в языке:

if(a==5) {
doThings();
} else if(a==4) {
AllOfThem();
} else
doSomething();

Циклы – отличная вещь. Они делают нашу жизнь проще, а код – эффективнее. Есть только 2 вида цикло в Java: for, который работает конкретное заданное количество раз,и while, который будет работать пока выполняется некоторое определяющее условие.

for(int i=0;i < n;++i) {}
int i=0;
while(i < n) {
++i;
}

Массивы в Java – ряды данных одного типа. Для объявления нужно уточнить тип данных элементов, его название и размер в квадратных скобках, вот так. Напоминаю, что элементы перечисляются с 0.

int[] cats = new int[10];
int[] mice = {4, 8, 10, 12, 16};

Для того, чтобы получить длину массива, используем встроенную функцию length, а для получения некоторого элемента мы помещаем его порядковый номер в квадратные скобки рядом с названием массива.
Также есть способ объявить динамический массив, размер которого можно изменить после объявления. Это делается с помощью ArrayList, вот так:

ArrayList catNames = new ArrayList();

У этого модуля свои методы, например size() возвращает размер массива, add() добавляет элемент, и, таки образом, меняет размер массива, а remove() удаляет элемент.
Давайте же создадим наш собственный проект на Java. Это будет полноценное десктоп-приложение, и оно будет о кофе. Приступим! Есть два важных класса, которые нужно запомнить: JFrame и JPanel. Они отвечают за рамку окна (ну ту часть с кнопкой “закрыть”), и за содержимое окна соответственно. Для того, чтобы сообщить компилятору, что мы будем использовать эти два класса, прибавим суффикс extends. Учитывая тот факт, что панель без рамки существовать не может, помещаем класс JPanel внутрь класса JFrame. Для настройки нашего окна мы должны написать немного обязательного кода, который отвечает за размер окна, его цвет, видимость, наличие кнопок на рамке, в главной функции. Вот что у нас должно получиться:

Код
Результат
public class Main extends JFrame {
public static class Main1 extends JPanel {}
public static void main(String[] args) {
JFrame f = new JFrame("coffee like java");
f.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
Main1 m = new Main1();
f.add(m);
f.setSize(900,700);
f.setResizable(false);
f.setVisible(true);
} }
