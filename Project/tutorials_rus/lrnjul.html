The Julia tutorial

Julia – достаточно нередкое имя. Возможно среди ваших знакомых есть Julia, но знали ли вы, что существует такой язык программирования? Да-да, именно так, Julia – это научный, новый и быстрый язык программирования, в основном используемый для комплексных математических вычислений. Он достаточно легко учится, синтаксис удобен для чтения, а скорость выполнения программ сравнима с С. Он берёт лучшие качества различных языков и собирает их воедино с широчайшими математическими дополнениями, таким образом получая идеальный язык для анализа данных. Ну, во всяком случае, это ожидается от языка.
Для того, чтобы начать работать с Julia, нам понабится компилятор, доступный для скачивания для всех платформ на официальном сайте.

Компилятор

Для установки Julia в Jupyter Notebook, нужно зайти в терминал Julia, который появится после установки, и ввести:

Pkg.add(“IJulia”)

Подобный код используется для установки дополнений и библиотек.
Для программистов уже стало традицией начинать программировать на новом языке с вывода в консоль сообщения “Hello, World!”. Здесь всё достаточно понятно.

Код
Результат
println("Hello, World!")

Стоит заметить, что в конце строки нет точки с запятой. Теперь о принятии пользовательского ввода. Есть много разных способов это сделать, но рекомендуется использовать функцию input(), с помощью которой также можно выводить сообщения перед вводом, вот так.

name = input("What is your name? ");

Переменные при изучении Julia не должны стать проблемой, учитывая тот факт, что их почти нет. При объявлении новой, необязательно уточнять тип. Сам компилятор знает, о каком типе речь, и его можно узнать с помощью функции typeof().

Code
Results
a = 2
typeof(a)

Так же объявляются и массивы, в случае с которыми элементы ставятся в квадратные скобки.

data = [1, 2, 3, 4, 5]

Копнём чуть глубже и объявим динамический массив, размер которого можно изменить после объявления. Здесь тип всё-таки уточняется.

data = Int64[];

А для того, чтобы добавить элемент, испльзуем push!(arrname, data).

Код
Результат
push!(data, 1)

Теперь поговорим об условиях. Их основное назначение – выполнить действие если выполняется условие. Работают они следующим образом.:

if condition
   action
elseif (condition && condition || condition)
   action
else
   action
end

Да, elseif – отдельный оператор в Julia, и работает он как else if. && используется как логическое «И», а || - как логическое «ИЛИ».
Действия, кстати, так размещены не просто так. Всё из-за правила о пробелах слева. Из-за того, что здесь не используются фигурные скобки для обозначения блоков кода, его границы уточняются этими пробелами. Один таб = один уровень. Во избежание ошибок рекомендется ставить end в конце каждого блока.
Теперь о такой прекрасной вещи как циклы. Они делают нашу жизнь проще, код – лучше, а кофе - крепче. Ну, насчёт последней части это у кого как, но удобство чтения и качество кода они точно улучшают. Есть 2 типа циклов: while, который выполняет код, пока выполняется соответствующее условие, и for, который выполняет код конкретное заданное количество раз.

while condition
   action
end
for i = 1:5
   action
end

Стоит заметить, что в примере с for двоеточием разделены начальная и конечная значения диапазона. В Julia, функции объявляется таким образом (правила о пробелах слева по-прежнему применяются):

function f(x, y)
   x + y
end

Для того, чтобы функция возвращала что-либо, не нужно менять кодовое слово. Достаточно просто добавить утверждение return в конце блока.

function g(x, y)
return x * y
   x + y
end

Несмотря на то, насколько недавно был выпущен этот язык, у Julia уже достаточно большое количество доступных библиотек и фреймворков, с которыми можно поэкспериментировать. Поэтому я желаю удачи в дальнейшем программировании, потому что она здесь точно пригодится.
