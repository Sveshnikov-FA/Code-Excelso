С++ - это язык программирования общего назначения, который в основном используется для создания интерфейсов, разработки десктоп, веб приложений и игр. Несмотря на то, что его производительность может быть ниже некоторых других языков, С++ любят за эффективность кода (не на уровне python, разумеется, но всё же), и возможности стандартной библиотеки, близкие к бесконечным.
Перед тем, как мы начнём программировать, мы должны убедиться, что у нас установлено всё необходимое. Нам будет нужен компилятор: для пользователей систем MacOS и UNIX это GNU C++, а для пользователей Windows - Microsoft Visual C++. (Как правило, он уже есть на новых компьютерах). MinGW подходит для всех платформ. Также нам нужна среда разработки, роль которой может выполнить Eclipse для С++.

У C++ свой набор переменных. Это char, одиночный символ, такой как ‘A’ или ‘%’; int, short и long – целые цисла соответствующей длины; float и double – действительные числа, где double имеет более широкий диапазон допустимых значений; Bool, тип, принимающий только одно из двух значений: либо true, либо false;

Условия в C++ объявляются достаточно просто. Если выполняется условие, то выполняется и действие.

Циклы – отличная вещь. Они делают нашу жизнь проще и наполняют её счастьем. Если быть точнее, то они помогают нам выполнять одни и те же действия несколько раз. В C++ 2 типа циклов: for, который всегда будет выполнен конкретное, заданное нами количество раз, и while, который будет выполняться пока выполняется заданное условие.

while(условие) {
//цикл выполняется, пока выполняется условие
}
for ( int I =0; i <,5; ++i ) {
//цикл выполнится 5 раз
}

Утверждения контроля циклов помогают нам изменить ход выполнения цикла. В С++ существуют 2 типа утверждений: break, полностью прерывающий итерацию цикла, и continue, игнорирующий код после этого и сразу переходящий на следующий круг.

while (a < 20) {
if (a==15) {
continue;
}
If (a==18) {
break;
}
a=a + 1;
}

Массив – это последовательность однотипных элементов, размещённых в ячейках памяти компьютера, достигаемых указанием индекса элемента в квадратных скобках. Они объявляются указанием типа содержимого, названием, размером в квадратных скобках, оператором присвоения, и элементами, перечисленными в фигурных скобках через запятую.

int foo [5]={ 16, 2, 77, 40, 12071 };

Также существуют динамические массивы, размер которых можно изменить после объявления - vector. Для того, чтобы их использовать, сначала нужно импортировать соответствующую библиотеку. Затем, для объявления мы сначала пишем, что это вектор, затем тип элементов в угловых скобках, затем название. Также после названия можно написать размер в скобках.

vector < int> myvector;

Для того, чтобы добавить элемент в конец вектора, используем функцию push_back, для удаления - pop_back.

myvector.push_back (100); myvector.pop_back();

В С++ можно создавать свои функции. Для объявления мы сначала пишем тип возвращаемых данных (int, char, или void, если мы не хотим ничего возвращать), затем название, затем аргументы, то есть значения, посылаемые в функцию из места вызова, затем действия в фигурных скобках. Для того, чтобы они были выполнены, необходимо вызвать функцию, написав сначала название, а затем - пересылаемые данные в скобках.
