The C++ tutorial

С++ is a general-purpose programming language, which is mainly used for numeric computing, marking up graphic user interfaces, game and web development. Like all languages, it has its fair share of advantages and disadvantages. On the latter side, it underperforms quite a bit compared to the other units on the lineup, but don’t click away just yet: there are many reasons as to why you should pick it up. C++ is well-known for its code’s efficiency (Of course, it’s not the ultra-efficient Python, but it’s definitely out there), and the inclusive standard template library, which, if you don’t know what that is, means that the range of possibilities of the language is remarkably wide.

Now, if you want to start coding properly, you’ll have to make sure you have all the necessary things installed. If your main objective is simple numeric programming, then an Online IDE will do, but if you really want to get into it then follow along the checklist: for MacOS and UNIX-based OS users, you need to install the GNU C++ compiler, the magic thing that tells the computer what to do with your code, and for Windows users, you need to check if you have Microsoft Visual C++ installed (usually it comes pre-installed on new computers) which will serve the purpose of the compiler. As for the IDE, the application that will send the code to the compiler and print out the results, I would recommend Eclipse CDT. After you’ve opened a fresh, brand-new C++ file, it’s time to start coding. It’s a good tradition to start off with a program that prints out “Hello, World!” so let’s do exactly that.

#include using namespace std; int main() { cout << "Hello World!"; }

So, half of the code above is auto-generated, and the other half is questionably abbreviated, so let’s break it down. #include<> defines the specific part of the standard template library you want to… include in the project, which means if you include the iostream part you will be able to use the syntax correlated to the iostream part, and to use the syntax for, say, vectors, you need to put an include statement in the beginning of the file. Namespace is, well, a space of certain names which you can reference further in the document. You can define your own namespace in which you put in global variables, so, to reference these variables without copying all of them in every project file, you can just write using namespace N; And std is a namespace pre-defined in C++, which helps shorten the code and make it far more readable, main() is the general function, akin to other main() voids and ints in other languages. Cout is the syntax for output, you put << signs every time you need to change the type, like here:

cout << N << ” ” << X;

So, unlike many other languages, in C++ you don’t use the + signs to add spaces. As usual, to output a number, you don’t need to use quotation marks.

Cout << 0;

C++ has its own set of variables, like any self-respecting programming language. Those include char, which represents a single character, such as ‘A’ or ‘%’; int,short and long; float and double, which represent real values, such as 3.14 with various levels of precision, depending on which type you want to choose, on double the precision is higher or equal to that of float’s; Bool, a type which represents a state: either true or false;

Another important basic step to touch upon is user input. It’s done in a quite similar way to the output, except the command is cin and instead of using <<, we use>> instead:

cin>>a>>b;

Conditionals in C++ are defined in a simple manner as well. You start off with the first if statement. It goes like this: if(condition) action. Conditions are set accordingly to the variable type, so if you need to compare a variable and a number, you use if(N==5), so double equation marks, if you want to check if a variable fits the value, then use triple equation marks. If you want to check if the Boolean is true, simply put if(boolean). If you want to check if the opposite is true, then use the exclamation point like this:

if(!(N==5))

or

if(N!=5)

or

if(!boolean)

Moving onto the action performed if said statement is true, if there is only one action, then there is no extra syntax to add, just put the action afterwards, and if there is more than one, then put said actions in {} brackets. But what if the condition is not true and you need something to cover for that? Well, C++ got you covered, because they implemented an additional else syntax. Basically, if the condition is false, you put an else statement with the same syntax as if. And yes, you can put in a condition after else.

Loops are great. They make our life better and fill it with joy. But actually they help us with repeating similar actions over and over again. There are two types of loops in C++: for and while. When we define a for loop we mention the initial value of the loop control variable [LCV], the condition for the continuation for the loop and the step of the LCV. And when we define a while loop we only put a condition for the continuation of the loop, and nothing else.

while(condition) { statement(s); }

for ( init; condition; increment ) { statement(s); }

Loop control statements help change the loop execution from its normal sequence. There are several kinds of statements: the break statement, which completely stops the execution of the loop and the continue statements which leads the loop to skip over the rest of its body for this iteration and start over the next one.

while (a<20) {

if (a==15) {

a=a + 1;

continue;

}

If (a==18) {

break;

}

a=a + 1;

}

An array is a series of elements of the same type placed in certain memory locations which can be accessed by adding an index to a unique identifier. You define an array by stating the type of elements it contains, its size in ph brackets, and you can also set its contains initially in curly brackets.

int foo [5]={ 16, 2, 77, 40, 12071 };

Elements in arrays are accessed by putting the index of the element, the first one being labeled 0, in ph brackets after the array name. There are also instances of dynamic arrays, the size of which can be modified after declaration. They are implemented in C++ with the vector type. To properly use vectors, you need to include in your C++ project, the same way we did above with . To define a vector, you need to state that it’s a vector, put the type of contains in angled brackets and then the name of the array.

vector myvector;

To add an element at the end of a vector and thus resizing it, you use the push_back function and to remove that element, use the pop_back function.

myvector.push_back (100); myvector.pop_back();

As you remember, the basic C++ project always includes int main(), which is the main function. Well, you can create your own functions, it’s just that to actually execute it you have to call it from the main int. Some functions return values, meaning that you can assign a variable to a function call which will result in the said variable being equal to the whatever value you return in the function like that: To define a function you put in the type of the return value (int, string) or void if you don’t neccesarily want to return something, then the function name, then you put in the arguments in parentheses, which are equal to the values you send from the main void. Then you put the actions to do in the function in curly brackets like that:

void woo() {

cout<< "This function will not return anything";

}

char ph[9]={ '1', '2', '3', '4', '5', '6', '7', '8', '9'};

Let’s break it down. Turn signifies whose players’ turn it is now, I is the loop iterator, pick is the index of the chosen square, the one that the player wants to make a move on, and mark is a character, either X or O, depending on whose players’ that’s the turn of. To properly process the turn, we’ll use a while loop.

Now we put in the conditional to switch turns.

if(turn%2!=0) turn=1; else turn=2;

There is a way to make conditionals cooler though, and that’s with something called ternary operators. The single string of code below does the same exact thing that the 2 lines above do. The question mark works like then, and the colon works like else.

turn=(turn%2)?1:2;

Now we get the players choice of turn.

cin>>pick;

And now we specify the mark. Again, we can use either the standard conditional or the ternary one.

if (turn==1) mark=’X’; else mark=’O’;

or

mark=(turn == 1) ? 'X' : 'O';

Now we actually play our turn. Let’s make a conditional that checks if the chosen spot is available for playing. If it’s not, the program will print out a message and reset the turn.
