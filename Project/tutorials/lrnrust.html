The Rust tutorial

Rust is something that appears on metal after a certain period of erosion, usually on objects that were around for a long time and served their purpose properly. Well, Rust is also a name of a programming language, which too is, coincidentally, trusty and heavily industrial. Most of the time it’s used for developing systems software and complex numeric operations. There are many ways to use this language, but if readability is an important factor for your coding experience, this language is not for you. Rust is, indeed, notorious for its syntax, but it definitely makes up for it with its efficiency (in terms of code length to result).

To start working with Rust you have to make sure you have all the necessary things installed, like the compiler, which you can get on the official Rust website and an IDE, for the role of which IntelliJ IDEa will suffice.

It’s a customary tradition for programmers to start learning a new language off by printing out “Hello, World!” in the console, so let’s get right to it!

fn main() {

println!("Hello World!");

}

It’s quite easy to break this code down: fn is the basic definition of a function or void in this language, main() being the – you guessed it – main function, the one where all the magic happens, the parameters for functions are defined in parentheses, the contains of the functions are put in curly brackets, to print things out you can use println! And put the thing you want to print out in parentheses and the string of text in brackets. Also it’s important to note the necessary semicolon at the end of the line. To declare an array of elements you make it changeable and set the type of elements and its size in square brackets and assign it elements, also in square brackets.

This might come as a shock, but you can also declare variables to print out in Rust. To be fair, there really aren’t any types to remember: all variable declarations start with code word let, whether it’s a real, floating point number, string or boolean. Basic numeric operations work as follows:

fn main() {

let sum = 5 + 10; let difference = 95.5 - 4.3; let product = 4 * 30; let quotient = 56.7 / 3 by 2.2; let remainder = 43 % 5; let mut array: [i32; 3] = [0; 3];

}

To get user input, use read_line with stdin() like this:

let mut s;

stdin().read_line(&mut s;)

Rust, just like any other language, has loops (or cycles), which are simple to understand: for, which is the cycle in which you have to directly set the range and step of the loop iterator and while, which works under the condition that a certain statement is true, where ranges and statements are defined in parentheses. But you have most likely never seen the mysteriously called loop loop, which is an official implementation of an infinite loop. They work as shown below:

loop {

println!("Loop forever!");

}

while (!done) {

println(“Loop for a while!”);

}

for (x = 0; x < 10; x++) {

println(“Loop in some range!”)

}

The concept of an infinite loop might seem pointless to you, but let me introduce you to break and continue statements. Break stops the cycle completely moving onto the next line of code, and continue moves on to the next iteration of the cycle.

Conditionals are the universal way to complete action if statement is true. There are also ways to execute certain functions if said statement is false by using the code word else like this:

if n < 10 && n > -10 {

println!(", and is a small number, increase ten-fold");

10 * n

} else {

println!(", and is a big number, half the number");

n / 2

}
